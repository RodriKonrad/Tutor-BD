Ayuda — Uso de la aplicación

Descripción

Esta aplicación proporciona una interfaz de chat simple basada en Blueprints y plantillas Jinja. La página principal permite enviar mensajes y recibir respuestas simuladas (eco).

Cómo usar

  1. Ve a la página Inicio.
  2. Escribe tu mensaje en el campo inferior y pulsa Enter o el botón Enviar.
  3. Verás el mensaje en la conversación; actualmente la aplicación devuelve una respuesta simulada "Echo: ...".

Atajos y accesibilidad

  • Enter: enviar mensaje (sin Shift).
  • El chat mantiene el foco en el campo de entrada tras enviar.
  • La zona de mensajes usa roles ARIA para mejorar la compatibilidad con lectores de pantalla.

Integración con backend

Ahora mismo las respuestas son locales y simuladas por `static/js/chat.js`. Para conectar con un backend real puedes:

  • Enviar mensajes a una ruta POST (`/api/message`) y devolver JSON con la respuesta.
  • Usar WebSockets (por ejemplo `flask-socketio`) para comunicación en tiempo real.

Solución de problemas

  • Si no ves estilos, asegúrate de que `static/css/style.css` está cargado y recarga la página (Ctrl+F5).
  • Si el JavaScript no responde, abre la consola del navegador y revisa errores en `chat.js`.
  • Verifica que la aplicación Flask está corriendo: `python run.py`.

Contacto

Para más ayuda, abre un issue en el repositorio o contacta al mantenedor del proyecto.
